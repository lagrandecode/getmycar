# App Icon Directory

## Instructions

1. **Place your source icon here**: 
   - Name it: `app_icon.png`
   - Size: 1024x1024px recommended
   - Format: PNG with transparency
   - Should contain: Your logo with white symbol, "GSTMYCAR" text, and purple background

2. **After placing the icon**, run:
   ```bash
   cd /Volumes/T7/findmycar/app
   flutter pub get
   flutter pub run flutter_launcher_icons
   ```

3. This will automatically generate all required icon sizes for both iOS and Android.

## Icon Requirements

- **Format**: PNG
- **Size**: 1024x1024px (minimum)
- **Background**: Deep purple (#6B46C1 or similar)
- **Content**: White diamond symbol with triangle cutout + "GSTMYCAR" text
- **Transparency**: Optional, but recommended for better integration
